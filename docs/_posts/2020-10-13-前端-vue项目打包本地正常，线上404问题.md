vue线上404问题记录

--- 

***具体问题描述:***

vue 本地打包， 测试打包后的文件运行正常。 将这些文件放到正式环境上， 点击页面跳转的时候直接报 404 。

---

***解决办法：***

将 route 配置的 model 改为 hash(原本配置可能是history导致的).

具体代码如下:

```javascript
文件位置: router/index.js

改变前：
const router = new VueRouter({
  mode: 'history',
  base: process.env.BASE_URL,
  routes
})

改变后：
const router = new VueRouter({
  mode: 'hash',
  base: process.env.BASE_URL,
  routes
})
```
