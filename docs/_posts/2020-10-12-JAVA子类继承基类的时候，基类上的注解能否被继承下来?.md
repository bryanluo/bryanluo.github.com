

***在子类继承基类（父类）的时候，　基类上一些特殊的属性（如：　注解）能否被继承下类？*** 

答案：不能

***验证过程如下:***

1. 自定义一个注解

```java
package com.jiang.annotation;

import java.lang.annotation.*;

/**
 * @author shijiang.luo
 * @RESOURCE feign-encoder
 * @createBy 2020/10/12
 */
@Documented
@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.TYPE)
public @interface T {

    String message() default "hello";

}

```

2. 编写一个基类(父类)

BaseEntity:

```java
package com.jiang.entity;

import com.jiang.annotation.T;
import lombok.Getter;
import lombok.Setter;

/**
 * @author shijiang.luo
 * @RESOURCE feign-encoder
 * @createBy 2020/10/12
 */

@T(message = "这里是基类")
@Setter
@Getter
public class BaseEntity {

    private String desc;


}
```

3. 编写一个子类(Person) 并且继承基类(BaseEntity)

Person:

```bash
package com.jiang.entity;

import lombok.AllArgsConstructor;
import lombok.Getter;
import lombok.NoArgsConstructor;
import lombok.Setter;

/**
 * @author shijiang.luo
 * @RESOURCE feign-encoder
 * @createBy 2020/10/12
 */
@Setter
@Getter
@AllArgsConstructor
@NoArgsConstructor
public class Person extends BaseEntity{

    private String name;
    
    private Integer age;
}

```

4. 测试

```java
package com.jiang;

import com.jiang.annotation.T;
import com.jiang.entity.BaseEntity;
import com.jiang.entity.Person;
import lombok.extern.slf4j.Slf4j;
import org.junit.Test;

/**
 * @author shijiang.luo
 * @RESOURCE feign-encoder
 * @createBy 2020/10/12
 */
@Slf4j
public class BaseConstructTest {

    @Test
    public void test() {

        BaseEntity baseEntity = new BaseEntity();
        T t1 = baseEntity.getClass().getAnnotation(T.class);
        log.info("BaseEntity Annotation : {}", t1.message());

        Person person = new Person();
        T t2 = person.getClass().getAnnotation(T.class);
        log.info("Person Annotation: {}", t2.message());
    }
}
```

***总结***
注解是不能被继承下来的。
